## Getting Started



### Installation




1. Install NPM packages
   ```sh
   npm install
   ```
2. Enter the schema.sql and seeds2.sql file paths into mysql
   ```js
   mysql> source db/schema.sql
   ```
   ```
    mysql> source db/seeds2.sql
   ```
3. Run nodemon server.js
    ```
    nodemon server.js
    ```
